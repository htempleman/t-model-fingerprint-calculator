Step 2 — Feature Pairs (one feature per row)

⚠ Elasticity Threshold Prerequisite

Before entering any feature pairs, confirm that all corresponding features have been assessed against the T-Model friction ridge skin elasticity thresholds:

  • Linear displacement: The distance between corresponding features in the latent and exemplar must not differ by more than 20% of the measured distance. Use a millimeter ruler to measure and confirm, if needed.
  • Rotational displacement: The directional difference between corresponding features must not exceed 10 degrees. Use a protractor to measure and confirm, if needed.
  • Features that exceed either threshold are discrepancies, not correspondences — they must be excluded from the calculation entirely and should not be entered below.
  • Connectivity disagreements (one impression shows an ending ridge, the other a bifurcation, at the same spatial location) that are within spatial tolerance should be entered with a Satisfactory agreement rating.

Document all elasticity threshold measurements in the case record before proceeding. See Chapter 8 for full guidance.

Measuring Linear Displacement

  1. Identify the corresponding feature pair in both the latent and exemplar impressions.
  2. Select a consistent reference point on each feature — for example, the tip of an ending ridge or the junction point of a bifurcation.
  3. Identify a second reference point in each impression — ideally the nearest Level 2 neighbor used for the intervening ridge count.
  4. Using a millimeter ruler, measure the distance between the two reference points in the latent impression. Record this measurement.
  5. Measure the same distance between the corresponding reference points in the exemplar impression. Record this measurement.
  6. Calculate the percentage displacement:
    % displacement = (measured distance − reference distance) ÷ reference distance × 100
    Where the reference distance is the measurement in the exemplar (the known, controlled impression). The result will be positive for stretch and negative for compression — both directions are subject to the 20% maximum threshold.
  7. If the result exceeds 20%, the feature fails the linear displacement threshold. It is a discrepancy and must be excluded from the calculation. Do not enter it below.
  8. If the result is 20% or less, the feature is within tolerance. Proceed to assess rotational displacement.

Measuring Rotational Displacement

  1. Identify the corresponding feature pair in both impressions.
  2. Establish the directional axis of the feature in each impression:
    — For an ending ridge: the direction the ridge tail points.
    — For a bifurcation: the direction of the stem before it divides.
    — For a dot: rotational displacement is not applicable.
  3. Using a protractor, measure the angle of the feature's directional axis relative to a consistent reference direction (e.g. the horizontal axis of the print) in the latent. Record the angle.
  4. Measure the same angle in the exemplar. Record the angle.
  5. Calculate the difference between the two angles.
  6. If the difference exceeds 10 degrees, the feature fails the rotational displacement threshold. It is a discrepancy and must be excluded from the calculation. Do not enter it below.
  7. If the difference is 10 degrees or less, the feature is within rotational tolerance and may be included in the calculation.

Note: A feature that passes both the linear and rotational threshold tests is eligible for inclusion. A feature that fails either test is a discrepancy and must be excluded entirely — there is no partial credit for a feature that almost satisfies either threshold. Document all measurements in the case record.
